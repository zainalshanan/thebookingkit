SlotKit provides 21 React components built on shadcn/ui, react-day-picker, react-big-calendar, and react-hook-form. Following the shadcn/ui convention, you **copy the source code** into your project and own it completely.

## Installation

Components live in `registry/ui/src/components/`. Copy the ones you need into your project:

```bash
# Copy a component
cp registry/ui/src/components/booking-calendar.tsx src/components/slotkit/

# Copy the cn() utility (used by all components)
cp registry/ui/src/utils/cn.ts src/components/slotkit/utils/
```

## Dependencies

Components depend on these npm packages:

| Package | Used by |
|---|---|
| `react-day-picker` | BookingCalendar |
| `react-big-calendar` | AdminScheduleView |
| `react-hook-form` | BookingQuestions, ManualBookingForm, WorkflowBuilder |
| `date-fns` | Date formatting |
| `@slotkit/core` | Hooks (useAvailability, useProvider) |

## Customization

All components accept `className` and `style` props. They use the `cn()` utility for class merging (compatible with Tailwind CSS):

```tsx
<BookingCalendar
  className="rounded-lg shadow-md"
  style={{ maxWidth: 400 }}
  {...otherProps}
/>
```
