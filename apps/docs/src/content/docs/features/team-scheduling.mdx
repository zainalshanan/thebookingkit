title: Team Scheduling

Team scheduling enables organizations with multiple providers to manage bookings across a team.

## Assignment strategies

### Round-robin

Distributes bookings evenly across team members. Considers each member's existing booking count to maintain balance.

```typescript
import { assignHost } from "@thebookingkit/core";
import type { TeamMemberInput, MemberBookingCount } from "@thebookingkit/core";

const teamMembers: TeamMemberInput[] = [
  {
    userId: "alice",
    role: "member",
    priority: 1,
    weight: 100,
    rules: aliceRules,
    overrides: aliceOverrides,
    bookings: aliceBookings,
  },
  {
    userId: "bob",
    role: "member",
    priority: 2,
    weight: 100,
    rules: bobRules,
    overrides: bobOverrides,
    bookings: bobBookings,
  },
];

// Track booking counts for load balancing
const bookingCounts: MemberBookingCount[] = [
  { userId: "alice", confirmedCount: 5 },
  { userId: "bob", confirmedCount: 7 },
];

const result = assignHost(teamMembers, bookingCounts, "round_robin");
// Returns: { hostId: "alice", reason: "alice selected via round-robin (load balanced)" }
// Alice gets assigned because Bob has more bookings
```

### Collective

All team members must be available for the slot. Used for group interviews, panel discussions, or meetings requiring all participants.

```typescript
import { getTeamSlots } from "@thebookingkit/core";
import type { TeamMemberInput } from "@thebookingkit/core";

const teamMembers: TeamMemberInput[] = [
  {
    userId: "alice",
    role: "member",
    priority: 1,
    weight: 100,
    rules: [mondayFriday9to5],
    overrides: [],
    bookings: aliceBookings,
  },
  {
    userId: "bob",
    role: "member",
    priority: 2,
    weight: 100,
    rules: [mondayFriday9to5],
    overrides: [],
    bookings: bobBookings,
  },
];

const slots = getTeamSlots(
  teamMembers,
  {
    start: new Date("2026-03-09T00:00:00.000Z"),
    end: new Date("2026-03-13T23:59:59.999Z"),
  },
  "America/New_York",
  { strategy: "collective", duration: 60 }
);

// Returns only slots where BOTH Alice and Bob are available
// If Alice is booked 2-3pm and Bob is booked 3-4pm,
// those slots are excluded from the results
```

### Managed

Organization-level event type templates with field locking. Admins create templates; members get auto-generated event types with locked fields.

```typescript
import { resolveManagedEventType, isFieldLocked, propagateTemplateChanges } from "@thebookingkit/core";
import type { ManagedEventTypeTemplate } from "@thebookingkit/core";

// Admin creates a template with some fields locked
const template: ManagedEventTypeTemplate = {
  title: "Team Consultation",
  durationMinutes: 30,
  bufferBefore: 0,
  bufferAfter: 15,
  customQuestions: [],
  priceCents: 5000,
  lockedFields: [
    { field: "title", locked: true },
    { field: "durationMinutes", locked: true },
    { field: "description", locked: false },  // Members can customize
    { field: "priceCents", locked: false },   // Members can set own price
  ],
};

// Check what members can and can't modify
isFieldLocked(template, "title");            // true
isFieldLocked(template, "durationMinutes");  // true
isFieldLocked(template, "description");      // false
isFieldLocked(template, "priceCents");       // false

// When admin updates the template
const updatedTemplate: ManagedEventTypeTemplate = {
  ...template,
  durationMinutes: 45,  // Changed from 30 to 45 minutes
};

// Propagate changes (locked fields only)
const updates = propagateTemplateChanges(updatedTemplate, memberEventTypes);
// All members' duration changes to 45 minutes
// Members' custom prices are NOT overwritten
```

### Fixed

Direct booking with a specific team member. No automatic assignment.

## Team slots

Complete example with all team members:

```typescript
import { getTeamSlots } from "@thebookingkit/core";
import type { TeamMemberInput } from "@thebookingkit/core";

const teamMembers: TeamMemberInput[] = [
  {
    userId: "alice",
    role: "admin",
    priority: 1,
    weight: 100,
    rules: [
      {
        rrule: "RRULE:FREQ=WEEKLY;BYDAY=MO,TU,WE,TH,FR",
        startTime: "09:00",
        endTime: "17:00",
        timezone: "America/New_York",
      },
    ],
    overrides: [],
    bookings: aliceBookings,
  },
  {
    userId: "bob",
    role: "member",
    priority: 2,
    weight: 100,
    rules: [
      {
        rrule: "RRULE:FREQ=WEEKLY;BYDAY=MO,TU,WE,TH,FR",
        startTime: "10:00",
        endTime: "18:00",
        timezone: "America/New_York",
      },
    ],
    overrides: [],
    bookings: bobBookings,
  },
];

const slots = getTeamSlots(
  teamMembers,
  {
    start: new Date("2026-03-09T00:00:00.000Z"),
    end: new Date("2026-03-13T23:59:59.999Z"),
  },
  "America/New_York",
  { strategy: "round_robin", duration: 30 }
);

// Example result
console.log(slots[0]);
// {
//   startTime: "2026-03-09T14:00:00.000Z",
//   endTime: "2026-03-09T14:30:00.000Z",
//   localStart: "2026-03-09T09:00:00",
//   localEnd: "2026-03-09T09:30:00",
//   availableMembers: ["alice", "bob"]  // both members are free at this time
// }
```

## UI component

The `<TeamAssignmentEditor />` component lets admins configure team members and strategies:

```tsx
import { TeamAssignmentEditor } from "./components/team-assignment-editor";

<TeamAssignmentEditor
  members={teamMembers}
  strategy={currentStrategy}
  onStrategyChange={handleStrategyChange}
  onMembersChange={handleMembersChange}
/>
```
